# 🎓 Zrqa Academic Web 

Portal web akademik interaktif dan pusat informasi khusus untuk LAB di (Sistem Informasi, Telkom University). Proyek ini dirancang untuk memudahkan manajemen pengumuman kelas, akses repositori, dan modul pembelajaran dalam satu antarmuka yang modern dan responsif.

## 🚀 Teknologi yang Digunakan

Proyek ini dibangun menggunakan *stack* pengembangan *frontend* modern:
- **[React](https://react.dev/)** - *Library* UI utama
- **[TypeScript](https://www.typescriptlang.org/)** - Keamanan tipe data (*Type-safe*)
- **[Vite](https://vitejs.dev/)** - *Build tool* dan *dev server* super cepat
- **[Tailwind CSS v4](https://tailwindcss.com/)** - *Utility-first CSS framework* untuk *styling*

## ✨ Fitur Utama

- **Papan Pengumuman:** Menampilkan informasi tenggat waktu tugas, kuis, dan aktivitas kelas lainnya.
- **Profil Identitas:** Menampilkan detail mahasiswa (Nama, NIM, Peran).
- **Akses Cepat:** Tautan langsung menuju repositori proyek kelas dan modul pembelajaran terkait.
